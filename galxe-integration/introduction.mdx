## GraphQL Endpoint & Authentication

**API Endpoint**: `https://graphigo-business.prd.galaxy.eco/query`

```bash
curl -X POST https://graphigo-business.prd.galaxy.eco/query \
  -H "Content-Type: application/json" \
  -H "access-token: YOUR_ACCESS_TOKEN" \
  -d '{"query": "query { __typename }"}'
```

Get your access token from [Galxe Dashboard Settings](https://dashboard.galxe.com).

## Quick Example
