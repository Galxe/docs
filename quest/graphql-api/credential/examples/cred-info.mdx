### Query credential information

Arguments:

| Name     | Description       |
|----------|-------------------|
| `id`     | Cred id.          |

Request:

```graphql
query fetchCredInfo($cred_id: ID!){
  credential(id: $cred_id){
    id
    name
    description
    chain
    referenceLink
    itemCount
    lastUpdate
    lastSync
    lastSyncedBlock
    syncStatus
    credType
    credSource
    recurrence
    dimensionConfig
    curatorSpaceID
    commonInfo{
      participateEndTime,
      modificationInfo,
      seoImage
    }
  }
}
```

Variables:

```json
{
  "cred_id": 473720589215887360
}
```
