# Space Access

### Query campaign participants

Arguments:

| Name     | Description       |
|----------|-------------------|
| `id`     | Campaign hash id. |
| `pfirst` | Query limit.      |
| `pafter` | Query offset.     |

Request:

```graphql
query campaignParticipants($id: ID!, $pfirst: Int!, $pafter: String!) {
  campaign(id: $id) {
    id
    numberID
    numNFTMinted
    participants {
      participants(first: $pfirst, after: $pafter) {
        list {
          address {
            username
            avatar
            address
            email
            solanaAddress
            aptosAddress
            seiAddress
            discordUserID
          }
        }
      }
      participantsCount
    }
  }
}
```

Variables:

```json
{
  "id": "GCn45UjHXE",
  "pfirst": 1,
  "pafter": "-1"
}
```
